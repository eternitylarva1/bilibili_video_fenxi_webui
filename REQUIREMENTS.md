# B站视频爬虫 WebUI 需求文档

## 用户身份
- B站UP主
- 题材：杀戮尖塔游戏
- 数据规模：约1000条视频（筛选条件：播放>2万 或 >2000/小时）

## 核心目标
通过分析同题材爆款视频数据，指导选材和创作决策

---

## 一、数据分析需求

### 1.1 播放量分析
- **播放量分布** - 了解多少播放量算爆款
- **时长 vs 播放量关系** - 什么时长的视频更容易爆
- **异常值标记** - 标记特别高（>50万）的爆款视频

### 1.2 标题关键词分析
- **高频词统计** - 提取爆款标题中的关键词
- **关键词效果** - 每个关键词的平均播放量
- **模式发现** - 发现"以防你不知道"、"假如"等成功模式

### 1.3 发布时间分析
- **小时分布** - 找出最佳发布时段（17-18点、11-12点）
- **星期分布** - 工作日vs周末效果差异

---

## 二、功能需求

### 2.1 散点图（最核心）
- **变量组合** - X/Y轴可选：播放量、点赞数、时长(分钟)、发布天数
- **比例尺设置** - 用户可设置X/Y轴范围（最小-最大值）
- **对数刻度** - 支持对数刻度（播放量跨度大时用）
- **滚轮缩放** - 以鼠标为中心放大/缩小，鼠标下方的点保持位置不变
- **点击查看** - 点击点显示视频详情（标题、作者、播放量、时长）
- **自动范围** - 默认时长0-60分钟，天数0-90天，播放量P98分位

### 2.2 筛选系统
- **时长筛选** - <5分钟 / 5-15分钟 / 15-30分钟 / >30分钟
- **播放量筛选** - >10万 / >20万 / >50万 / >100万
- **标题搜索** - 关键词搜索
- **关键词筛选** - 从关键词云点击筛选

### 2.3 TOP榜单
- **列表/TOP视图切换**
- **TOP50排名** - 按播放量降序
- **TOP3高亮** - 金/银/铜牌样式
- **爆款标记** - >50万高亮显示

### 2.4 统计概览
- **卡片统计** - 总视频数、总播放量、总点赞
- **柱状图** - 播放量分布（可缩放高度）
- **环形图** - 时长分布
- **时间线图** - 发布时间分布

---

## 三、技术需求

### 3.1 性能
- 支持1000+条数据流畅渲染
- 图表重绘性能优化

### 3.2 交互
- 所有图表高度可调整（拖拽底部resize条）
- 图表最小100px，最大400px，默认180px
- 响应式布局，适配手机/平板/桌面

### 3.3 数据准确性
- 时长解析正确处理 HH:MM:SS 格式
- 排除录播合集等异常值
- 标尺从0开始（对数刻度除外）

## 六、数据洞察（杀戮尖塔题材）

### 6.1 时长与爆款率
| 时长 | 视频数 | >10万播放比例 |
|-----|-------|--------------|
| <5分钟 | 641条 | **91.4%** |
| 5-30分钟 | 293条 | 79.5% |
| >30分钟 | 67条 | 样本少但均值高 |

### 6.2 爆款标题模式
| 模式 | 数量 | 平均播放 |
|-----|------|---------|
| "以防你不知道" | 38条 | **29.4万** |
| "mod/模组" | 50条 | 27.2万 |
| "杀戮尖塔2" | 470条 | 26.8万 |
| "假如" | 78条 | 18.0万 |
| "进阶" | 28条 | 13.1万 |

### 6.3 发布时机
- **黄金时段**：17-18点、11-12点（发布量最大）
- **最佳时段**：17-18点（下班时间）
- **周末**：发布量较少但竞争也少

### 6.4 播放量分布
- 中位数：20.4万
- 75分位：30.4万
- 90分位：46.5万

## 八、Agent会话管理

### 可复用的会话（Resumable Sessions）

| Agent | Session ID | 用途 |
|-------|-----------|------|
| designer | des-2 | 分析散点图比例尺问题 |
| designer | des-1 | 改进WebUI交互和功能 |

**复用规则：**
- 同一项目中，相同任务的讨论应复用同一个 session
- 新任务或不同方向的任务应开新的 session
- session ID 一旦建立，在该任务结束前不应更换
